What is Git?

Git is a version control system. It tracks changes in your code, lets you go back in time, and makes teamwork easier.

  • Save checkpoints with git commit
  • Experiment safely using branches
  • See who changed what and when

Why do developers love Git?

Because it turns messy projects into clean, organized history. If something breaks, you can always roll back to a working version.

  • Undo mistakes quickly
  • Collaborate with your team
  • Keep your project history forever

What is GitHub Actions?

GitHub Actions is an automation engine inside GitHub. It can build, test, and deploy your code automatically whenever you push. For this project, we use it to deploy our static site to GitHub Pages.
