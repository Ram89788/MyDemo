What is Git?

Git is a distributed version control system. It tracks your code history so you can experiment, revert, and collaborate safely.

Why we use Git

  • Undo mistakes easily
  • Work in teams without conflicts
  • Keep a complete project history

Basic Git commands

git init
git status
git add .
git commit -m "first commit"
git push origin main

What is GitHub Actions?

GitHub Actions is an automation engine inside GitHub. It can build, test and deploy your project whenever you push code. For this demo, it deploys this static website to GitHub Pages.

YAML Workflow

The file .github/workflows/deploy.yml describes:

  • When to run (on push to main)
  • What jobs to execute
  • How to deploy to GitHub Pages
